<h1>Uma análise sobre como o TSE tem julgado a prestação de contas dos partidos</h1>

Esse projeto foi o primeiro submetido durante o Lede Program, curso de verão da Columbia sobre jornalismo de dados. 
De início, eu tinha como objetivo para meu primeiro projeto analisar o financiamento de campanha eleitoral para saber quais eram os principais doadores historicamente. Porém, a base de dados fornecida pelo TSE era muito suja, com muitos valores faltantes, além de enorme, e requeriria uma checagem adicional, cruzando com os PDFs das notas fiscais. Seria a bit too much for a first data project. 
Encontrei essa base de dados menor, melhor estruturada e menox complexa, sobre o resultado do julgamento das prestações de contas partidárias, o que é bem próximo da minha ideia inicial. Foi a primeira vez que tive contato com esse dataset.
