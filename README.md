<h1>Uma análise sobre como o TSE tem julgado a prestação de contas dos partidos</h1>

<h2>What is this</h2>

<p>Esse projeto foi o primeiro submetido durante o Lede Program, curso de verão da Columbia sobre jornalismo de dados.</p> 
<p>A ideia de fazer um projeto sobre o esse tema começou após a aprovação de um projeto de lei na Câmara dos Deputados do Brasil que queria criar uma mini reforma eleitoral para reduzir o valor das multas aplicadas aos partidos se a prestação de contas anual fosse rejeitada, assim como instituir a prescrição de um ano se o TSE não julgasse os balanços. Descobri então que havia uma base de dados do TSE sobre o resultado do julgamento das prestações de contas partidárias. Meu principal interesse era saber quais eram os partidos que mais acumulavam multas eleitorais e quais tinham tido as contas rejeitadas mais vezes. 

<h2>What I found</h2>
Minha primeira surpresa foi descobrir que a maior parte das prestações de contas não foi julgada ainda (mais de 60%). E que cinco partidos não enviaram os balanços para o tribunal - algo que ocorreu um total de 10 vezes entre 2010 e 2024. 
Também pude descobrir que dois partidos dentre os três autores do projeto de lei estão entre os que mais acumulam multas eleitorais, o que era uma hipótese inicial.
